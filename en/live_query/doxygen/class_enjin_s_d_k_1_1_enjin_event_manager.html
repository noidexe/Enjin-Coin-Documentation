Detailed Description

Template Parameters
T

Event Manager replacement model using C# Action (Delegate) model versus UnityEvent model. Abstract Action is used in the dictionary to provide support for elements beyond the SDK allowing this Event Manager to be used by clients in their applications for events outside of the Enjin SDK scope.

Similiarly, the Event Manager is provided with MonoBehavior support allowing it to be used with game objects.
